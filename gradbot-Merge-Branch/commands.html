Coding Your Robot


'side.setPower(level);'
Sets the power level of
the motor on the specified
side of the robot.
Side can be left or right.
Level can range from
0 to 100.

'await delay(number);'
Waits to move on to the
next command. Number is
the number of milliseconds
you would like the robot
to wait.

'laser.fire;'
Fires the specified laser
one time. Use part name to
specify which laser to fire from.
